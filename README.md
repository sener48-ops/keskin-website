# KESKİN

Mühendislik, makine üretimi ve doğal zeytin ürünleri firması için **sunucusuz (statik)** kurumsal tanıtım web sitesi. Saf HTML, CSS ve JavaScript ile yazılmıştır; herhangi bir derleme aracı veya backend gerektirmez.

## Bölümler
1. **Danışmanlık & Mühendislik** — proje/tasarım, süreç optimizasyonu, teknik danışmanlık
2. **Makine Üretimlerimiz** — kendi üretimimiz makineler
3. **Fason Makine & Yedek Parça** — CNC talaşlı imalat, yedek parça, tersine mühendislik
4. **Zeytinyağı & Zeytin Ürünleri** — natürel sızma zeytinyağı, sofralık zeytin ve yan ürünler

Ayrıca hero, istatistik şeridi, iletişim ve footer bölümleri içerir. Tam responsive (mobil/tablet/masaüstü), kurumsal endüstriyel tema.

## Çalıştırma
`index.html` dosyasına çift tıklayıp tarayıcıda açın. Sunucu gerekmez.

## Dosya Yapısı
```
index.html        # Tek sayfa, tüm bölümler
css/style.css     # Tema (CSS değişkenleri) + responsive
js/main.js        # Menü, smooth scroll, scroll-reveal
```

## Özelleştirme
İçerik placeholder'dır. HTML içinde `<!-- DEĞİŞTİR -->` yorumlarıyla işaretli alanları (telefon, e-posta, adres, sloganlar) ve görselleri kendi bilgilerinizle güncelleyin.
